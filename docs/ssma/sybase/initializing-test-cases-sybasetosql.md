# <a name="initializing-test-cases-sybasetosql"></a>テスト ケースの初期化 (SybaseToSQL)
このステップでは、新しいテストケースの初期情報を指定します。  
  
## <a name="parameters"></a>パラメーター  
**テストケース名**  
テストケースを識別する名前を入力します。  
  
**[作成日付]**  
今日の現在の日付は自動的に定義されます。  
  
**最終更新日**  
自動的に入力されます。変更しないでください。  
  
**テストケースの説明**  
テストケースの目的を特定するための追加情報を入力します。

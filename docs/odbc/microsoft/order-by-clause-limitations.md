# <a name="order-by-clause-limitations"></a>ORDER BY 句の制限事項
SELECT ステートメントに GROUP BY 句と ORDER BY 句が含まれている場合、ORDER BY 句には、結果セット内の列、または GROUP BY 句の式のみを含めることができます。

# <a name="getbyte-method-sqlservercallablestatement"></a>getByte メソッド (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  指定されたパラメーターの値を **byte** 値として取得します。  
  
## <a name="overload-list"></a>オーバーロードの一覧  
  
|名前|説明|  
|----------|-----------------|  
|[getByte (int)](../../../connect/jdbc/reference/getbyte-method-int.md)|パラメーターに渡されたインデックスを使用して、指定されたパラメーターの値を **バイト** 値として取得します。|  
|[getByte (java.lang.String)](../../../connect/jdbc/reference/getbyte-method-java-lang-string.md)|パラメーターに渡された名前を使用して、指定されたパラメーターの値を **バイト** 値として取得します。|

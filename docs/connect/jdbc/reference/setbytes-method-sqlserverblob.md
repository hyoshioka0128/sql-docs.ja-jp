# <a name="setbytes-method-sqlserverblob"></a>setBytes メソッド (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  渡された byte 配列を、渡された位置から BLOB に書き込み、書き込んだバイト数を返します。

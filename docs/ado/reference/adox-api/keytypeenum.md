# <a name="keytypeenum"></a>KeyTypeEnum
[キー](./key-object-adox.md)の種類 (プライマリ、外部、または一意) を指定します。  
  
|定数|値|説明|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|既定値。 キーは主キーです。|  
|**adKeyForeign**|2|キーが外部キーです。|  
|**adKeyUnique**|3|キーは一意です。|

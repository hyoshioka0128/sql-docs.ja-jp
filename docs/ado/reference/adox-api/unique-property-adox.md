# <a name="unique-property-adox"></a>Unique プロパティ (ADOX)
インデックスキーが一意である必要があるかどうかを示します。  
  
## <a name="settings-and-return-values"></a>設定と戻り値  
 **ブール** 値を設定して返します。  
  
## <a name="remarks"></a>解説  
 既定値は **False** です。  
  
 このプロパティは、既にコレクションに追加されている [インデックス](./index-object-adox.md) オブジェクトでは読み取り専用です。

# <a name="type-property-table-adox"></a>Type プロパティ (テーブル) (ADOX)
テーブルの種類を示します。  
  
## <a name="return-values"></a>戻り値  
 テーブルの種類を指定する **文字列** 値を返します。たとえば、"TABLE"、"SYSTEM TABLE"、"GLOBAL TEMPORARY" などです。  
  
## <a name="remarks"></a>コメント  
 このプロパティは読み取り専用です。
